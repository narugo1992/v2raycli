Installation
===================

v2raycli is currently hosted on PyPI. It required python >= 3.7.

You can simply install v2raycli from PyPI with the following command:

.. code:: shell

    pip install v2raycli

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/v2raycli@main
